# Elite Electron Client

A cross-platform desktop client for Transformice and DeadMaze, built on Electron.

## Development

```bash
npm install
npm run tsc
npm start
```

## Build

```bash
npm run buildlnx   # Linux AppImage
npm run buildwin   # Windows
npm run buildmac   # macOS
```

## Features

- Flash Player bundled and auto-installed
- Customizable UI themes (Dark, Amoled, Light)
- FPS overlay with position/size/color options
- Ping monitor
- Keyboard shortcut manager
- Settings export/import
- Cache management
